Installazione del driver

Per installare il driver, procedere come segue:
  1. Lanciare il file eseguibile "setup.exe"
  2. Seguire le istruzioni visualizzate dal programma setup.exe


NOTA:

    
  • Su piattaforme Windows di tipo Plug&Play, l'adattatore viene automaticamente riconosciuto e configurato. Interrompere l'installazione automatica da parte di Windows. 
  • Per installare il driver è necessario disporre di privilegi di amministratore.


Installazione manuale

L'installazione manuale consiste nell'installare il driver mediante il file di installazione (INF) anziché usare "setup.exe" menzionato in precedenza. Per installare il driver manualmente, aprire la Gestione periferiche, espandere la struttura Rete, selezionare "Dispositivo di controllo della famiglia Realtek FE/Gbe", fare clic con il tasto destro del mouse ed eseguire l'elemento del menu Update driver... [Aggiorna driver], quindi seguire le indicazioni per la posizione dove salvare il file INF.


NOTA:

  • L'installazione manuale è consigliata agli utenti esperti. Se non si possiede sufficiente dimestichezza, utilizzare il file "setup.exe" per installare il driver.


Installazione invisibile all'utente

Questa sezione descrive il processo di installazione del driver nascondendo la visualizzazione di finestre di dialogo sullo schermo dell'utente e senza necessità di azioni da parte dello stesso. Per eseguire un'installazione invisibile all'utente, lanciare il prompt di DOS ed eseguire

Installazione invisibile all'utente 

    C:>%driver_path%\setup.exe /s

Disinstallazione invisibile all'utente 
    C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"

Risoluzione dei problemi

In caso di problemi durante l'installazione del driver, seguire attentamente le istruzioni descritte nella relativa sezione per ripetere la procedura. Se il problema persiste, rivolgersi all'assistenza tecnica. Maggiori informazioni sono disponibili sul sito Web Realtek.
