Instalación de los controladores

Para instalar el controlador, haga lo siguiente:
  1. Abra el archivo ejecutable "setup.exe".
  2. Siga las instrucciones que se muestran en el programa setup.exe.


NOTA:

    
  • En plataformas Windows Plug and Play, el adaptador se identificará y configurará automáticamente. Detenga la instalación automática del sistema Windows. 
  • Para instalar el controlador deberá tener privilegios de administrador.


Instalación manual

La instalación manual del controlador puede realizarse utilizando el archivo (INF) en lugar del archivo "setup.exe" mencionado anteriormente. Para instalar el controlador manualmente, abra el Administrador de dispositivos, expanda la lista de adaptadores de red, seleccione "Controlador de la familia FE/GbE de Realtek", haga clic en él con el botón secundario del ratón y ejecute la opción "Actualizar controlador...". A continuación, siga las instrucciones para indicar la ruta de acceso al archivo INF.


NOTA:

  • La instalación manual es un método de instalación para usuarios avanzados. Si usted es un usuario genérico, utilice el archivo "setup.exe" para instalar el controlador.


Instalación silenciosa

Esta sección describe el proceso de instalación del controlador sin mostrar ningún cuadro de diálogo en la pantalla del usuario ni solicitar su intervención. Si necesita instalar el controlador de modo silencioso, abra una ventana de DOS y ejecute

Instalación silenciosa 

    C:>%driver_path%\setup.exe /s

Desinstalación silenciosa 
    C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"

Solución de problemas

Si encuentra algún problema durante la instalación del controlador, siga detenidamente las instrucciones descritas en la sección de Instalación e inténtelo de nuevo. Si aún necesita ayuda, póngase en contacto con nuestro departamento de soporte técnico. Para más información, acceda a nuestra página web.
