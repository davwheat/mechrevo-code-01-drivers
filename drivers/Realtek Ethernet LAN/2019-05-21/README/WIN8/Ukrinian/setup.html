Інсталяція драйвера

Щоб інсталювати драйвер, виконайте наведені нижче кроки

1.     Запустіть файл виконання «setup.exe»

2.     Дотримуйтеся інструкцій, що відображаються в програмі setup.exe


ПРИМІТКА.

  • На платформах Plug and Play ОС Windows ідентифікація та налаштування адаптера відбуваються автоматично. Зупиніть всі автоматичні інсталяції системи Windows.
  • Щоб інсталювати драйвер, ви повинні мати права адміністратора.
  • Якщо відображатиметься викривлений текст, використовуйте файл «AutoInst.exe», щоб повторно інсталювати/видалити драйвер.
    Файл «AutoInst.exe» діє як файл «Setup.exe».


Інсталяція вручну

Інсталяція вручну використовується для інсталяції драйвера за допомогою файлу інсталяції (INF), а не зазначеного файлу «setup.exe». Щоб інсталювати драйвер вручну, відкрийте диспетчер пристроїв, відкрийте деревовидний список мережевих адаптерів, поставте прапорець біля контролера «Realtek FE/Gbe Family Controller», клацніть правою клавішею миші, виконайте параметр меню «Оновити драйвер...», та дотримуйтеся напрямку, щоб вказати шлях до файлу INF.


ПРИМІТКА.

  • Інсталяція вручну — це метод інсталяції для досвідчених користувачів. Звичайним користувачам слід використовувати файл «setup.exe» для інсталяції драйвера.


Автоматична інсталяція

У цьому розділі описано процес інсталяції драйвера без відображення діалогового вікна на екрані користувача та без нагляду з боку користувача. Якщо потрібно виконати автоматичну інсталяцію драйвера, запустіть підказку DOS та виконайте команду

Автоматична інсталяція 

C:>%driver_path%\setup.exe /s


Автоматичне видалення 

C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"


Виправлення неполадок

У разі виникнення будь-яких проблем на етапі інсталяції драйвера уважно дотримуйтеся інструкції, що описано в розділі «Інсталяція», щоб виконати повторну інсталяцію. Якщо вам потрібна буде допомога, зверніться в відділ технічної підтримки. Детальна інформація доступна на веб-сайті.
