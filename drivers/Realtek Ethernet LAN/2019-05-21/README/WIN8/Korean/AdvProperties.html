고급 속성

사용자는 고급 속성을 운영 환경과 성능 고려사항에 따라 역동적으로 조절할 수 있습니다. 고급 속성의 설정을 조정하려면, [장치 관리자]를 열고 네트워크 어댑터 트리 목록을 확장한 다음 "Realtek FE/GbE 계열 컨트롤러"를 선택하고 마우스의 오른쪽 버튼을 클릭한 다음 [속성]의 메뉴 항목을 표시합니다. "Realtek FE/GbE 계열 컨트롤러" 항목이 표시되면 [고급]의 속성 시트를 선택합니다.


참고:

  • 고급 사용자의 경우 고급 속성 설정을 사용합니다. 일반 사용자의 경우 기본 값을 사용하시기 바랍니다.

  • 고급 속성을 부적절하게 설정할 경우 연결 오류 또는 성능 저하로 이어질 수 있습니다. 그와 같은 경우가 발생하면 이 절을 참조하여 적절한 설정을 선택하거나 기본 값으로 복원하십시오.
