Treiberinstallation

Gehen Sie zur Installation des Treibers folgendermaßen vor
  1. Öffnen Sie die selbstausführende Datei "setup.exe"
  2. Folgen Sie den Anweisungen, die das Installationsprogramm setup.exe Ihnen anzeigt


HINWEIS:

    
  • Plug & Play-Plattformen mit Windows identifizieren und konfigurieren den Adapter automatisch. Unterbrechen Sie bitte die automatische Installation von Windows. 
  • Für die Installation des Treiber müssen Sie über Administratorenrechte verfügen.


Manuelle Installation

Bei der manuellen Installation wird der Treiber über eine spezielle Installationsdatei (INF-Datei) statt über das oben beschriebene Verfahren mit der Datei „setup.exe“ installiert. Zur manuellen Treiberinstallation öffnen Sie den Geräte-Manager, erweitern die „Netzwerkadapter“-Liste (dazu auf das Pluszeichen davor klicken), klicken mit der rechten Maustaste auf Realtek FE/GbE-Controller und wählen im Kontextmenü den Eintrag Treibersoftware aktualisieren. Anschließend wählen Sie den Pfad zur INF-Datei anhand der Anweisungen auf dem Bildschirm („Auf dem Computer nach Treibersoftware suchen.“) aus.


HINWEIS:

  • Die manuelle Installation sollte nur von versierten Benutzern durchgeführt werden. Unerfahrene Benutzer verwenden bitte die Datei "setup.exe" zur Installation des Treibers.


Installation im Hintergrund

In diesem Abschnitt erfahren Sie, wie der Treiber ohne weiteren Benutzereingriff und Bildschirmmeldungen installiert werden kann. Um den Treiber im Hintergrund zu installieren, öffnen Sie die DOS-Eingabeaufforderung und führen

die Installation im Hintergrund aus 

    C:>%driver_path%\setup.exe /s

Deinstallation im Hintergrund 
    C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"

Fehlersuche

Falls bei der Installation des Treibers Probleme auftreten, wiederholen Sie die Installation unter genauer Beachtung der Anweisungen im obigen Abschnitt. Falls die Installation trotzdem fehlschlägt, wenden Sie sich an unsere Kundendienstabteilung. Weitere Informationen erhalten Sie auf unserer Website.
