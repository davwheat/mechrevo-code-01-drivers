歡迎使用Realtek RTL81XX全系列Fast/Gigabit Network Adapter


簡介

此使用說明包含了安裝 Realtek RTL81XX全系列Fast/Gigabit Network Adapter 所需要的資訊 或是於安裝時，可能遇到的問題及故障排除。


使用此文件

此使用手冊適用於安裝 Realtek RTL81XX系列 Fast/Gigabit Network Adapter於下列 Windows 作業系統：

Windows 8

  • NDIS 6.30 Miniport driver Windows 8(32-bit)
  • NDIS 6.30 Miniport driver Windows 8(64-bit)

Windows 7
  • NDIS 6.20 Miniport driver Windows 7(32-bit)
  • NDIS 6.20 Miniport driver Windows 7(64-bit)

Windows Vista
  • NDIS 6.0 Miniport driver Windows Vista(32-bit)
  • NDIS 6.0 Miniport driver Windows Vista(64-bit)

Windows XP
  • NDIS 5.1 Miniport driver Windows XP/Server 2003(32-bit)
  • NDIS 5.1 Miniport driver Windows XP/Server 2003(64-bit)
