מאפיינים מתקדמים

משתמשים יכולים לשנות באופן דינמי את המאפיינים המתקדמים בהתאם לסביבה התפעולית ולשיקולי הביצועים. לשינוי המאפיינים המתקדמים, פתח את מנהל ההתקנים, הרחב את עץ הרשימה מתאמי רשת, בחר באפשרות "הבקר המשפחתי Realtek FE/GbE", לחץ לחיצה ימנית על הפריט מאפיינים... כעת יוצגו מאפייני "הבקר המשפחתי Realtek FE/GbE" בחר בכרטיסייה מתקדם.


שים לב :

  • ההגדרות המתקדמות מיועדות למשתמשים מנוסים. אם אינך משתמש מנוסה, בחר בערכי ברירת המחדל.

  • הגדרה שגויה של מאפיינים מתקדמים עלולה למנוע התחברות או לפגוע בביצועים. במקרה כזה, קרא סעיף זה ובחר בהגדרות המתאימות או שחזר את ערכי ברירת המחדל.

Jumbo Frame

יחידת השידור המרבית (MTU) של רשת Ethernet היא באורך 1500 בתים. Jumbo Frame היא מסגרת שאורכה עולה על 1500 בתים. המשתמש יכול לבחור באפשרות זו כדי לשנות את גודל יחידת השידור המרבית בה תומך כרטיס הרשת.

6KB MTUקובע Jumbo Frame בגודל 6K בתים.
