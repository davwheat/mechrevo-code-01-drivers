Propriedades avançadas

Os utilizadores podem ajustar de forma dinâmica as propriedades avançadas de acordo com o sistema operativo e o desempenho pretendido. Para ajustar as propriedades avançadas, abra o Gestor de dispositivos, expanda a lista de Placas de rede, seleccione "Realtek série FE/GbE", clique com o botão direito do rato e abra o item Propriedades. Serão mostradas as propriedades da placa "Realtek série FE/GbE" e seleccione o separador Avançadas.


NOTA：

  • As propriedades avançadas apenas devem ser definidas pelos utilizadores mais experientes. Os outros utilizadores devem usar os valores predefinidos.

  • Uma incorrecta definição destas propriedades avançadas pode levar à falha na ligação ou a uma degradação do desempenho. Se tal acontecer, consulte esta secção para escolher a definição correcta ou proceda ao restauro das opções predefinidas.


Auto Disable Gigabit (PowerSaving) (Desactivar Gigabit automaticamente) (Poupança de energia)

Para atingir o objectivo de poupança de energia, o controlador pode desactivar automaticamente a função de Gigabit quando o cabo for ligado novamente.
Opções Descrição
Desactivar Desactiva a função Auto Disable Gigabit (Desactivar Gigabit automaticamente) (predefinição)
Re-Link, Battery (Ligar novamente, bateria) Ao usar a bateria, a função Gigabit é automaticamente desactivada quando o cabo é novamente ligado
Re-Link, Battery or AC (Ligar novamente, bateria ou a.c.) A função Gigabit é automaticamente desactivada quando o cabo é novamente ligado


NOTA：

  • (*) A função é suportada apenas pela placa Gigabit Ethernet.


Flow Control (Controlo do fluxo)

A placa de rede Realtek série FE/GbE pode parar temporariamente a transmissão de pacotes enquanto recebe pacotes de controlo de fluxo, ou enviar pacotes de controlo de fluxo para parar o lado da transmissão, de forma a adaptar a sua velocidade à velocidade no lado da recepção. O pacote de controlo de fluxo pára o lado da transmissão

Opções Descrição
Activar Activa o controlo do fluxo (predefinição)
Desactivar Desactiva o controlo do fluxo


Green Ethernet

É um modo de poupança de energia. A placa de rede Realtek série FE/GbE consegue detectar o comprimento do cabo de ligação para determinar a tensão de saída da transmissão e diminuir o consumo de energia.

Opções Descrição
Activar Activar a função Green Ethernet
Desactivar Desactivar a função Green Ethernet


NOTA：

  • (*) A predefinição varia de acordo com o produto. Consulte a folha de dados para definir a função.

Jumbo Frame (Estrutura jumbo)

A unidade máxima de transmissão (MTU, Maximum Transmit Unit) da Ethernet é de 1500 bytes. A estrutura jumbo é a estrutura cujo tamanho é superior a 1500 bytes. O utilizador pode usar esta opção para ajustar a unidade máxima de transmissão suportada pela placa NIC da Ethernet.

Opções Descrição
2KB MTU (MTU de 7 KB)Define a estrutura jumbo para 2 Kbytes.
3KB MTU (MTU de 3 KB)Define a estrutura jumbo para 3 Kbytes.
4KB MTU (MTU de 4 KB)Define a estrutura jumbo para 4 Kbytes.
5KB MTU (MTU de 5 KB)Define a estrutura jumbo para 5 Kbytes.
6KB MTU (MTU de 6 KB)Define a estrutura jumbo para 6 Kbytes.
7KB MTU (MTU de 7 KB)Define a estrutura jumbo para 7 Kbytes.
8KB MTU (MTU de 8 KB)Define a estrutura jumbo para 8 Kbytes.
9KB MTU (MTU de 9 KB)Define a estrutura jumbo para 9 Kbytes.
Disable (Desactivar) Desactiva a estrutura jumbo (predefinição)


NOTA:

  • Para activar a estrutura jumbo, é necessário activar também esta capacidade no outro lado da ligação.
