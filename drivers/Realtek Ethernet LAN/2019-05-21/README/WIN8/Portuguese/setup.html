Instalação do controlador

Para instalar o controlador, faça o seguinte
  1. Abra o ficheiro executável "setup.exe"
  2. Siga as instruções mostradas pelo programa setup.exe


NOTA:

    
  • Nas plataformas Windows Plug and Play, o controlador é identificado e configurado automaticamente. Páre a instalação automática por parte do sistema operativo Windows. 
  • Para instalar o controlador, tem de ter privilégios de administrador.


Instalação manual

A instalação manual permite-lhe instalar o controlador através do ficheiro INF em vez de usar o ficheiro "setup.exe" mencionado acima. Para instalar o controlador manualmente, abra o Gestor de dispositivos, expanda a lista de Placas de rede, seleccione "Realtek série FE/GbE", clique com o botão direito do rato e abra o item Actualizar controlador. Navegue até à pasta que contém o ficheiro INF.


NOTA：

  • A instalação manual deve ser usada apenas pelos utilizadores mais experientes. Os outros utilizadores devem usar o ficheiro "setup.exe" para instalar o controlador.


Instalação silenciosa

Esta secção descreve a instalação do controlador sem que seja mostrada qualquer caixa de diálogo no ecrã e sem que seja necessária a presença do utilizador. Se quiser instalar o controlador de forma silenciosa, abra a prompt do DOS e execute o comando seguinte

para uma instalação silenciosa 

    C:>%driver_path%\setup.exe /s

ou o comando abaixo para uma remoção silenciosa 
    C:>%driver_path%\setup.exe /s /f1".\uninstall.iss"

Resolução de problemas

Se tiver qualquer problema durante a instalação do controlador, siga atentamente as instruções descritas na secção Instalação para instalar novamente o controlador. Se continuar a precisar de ajuda, contacte o nosso departamento de assistência técnica. Para mais informações, visite o nosso web site.
